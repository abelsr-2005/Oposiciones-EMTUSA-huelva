# 📚 Repositorio de Preparación: Oposición Informática EMTUSA 2025

Este repositorio contiene los apuntes estructurados para la preparación de la Bolsa de Empleo de Informática de EMTUSA (2025). El contenido está estrictamente alineado con la convocatoria oficial e indexado para un repaso eficiente.

> **Nota de Auditoría:** El temario exige profundidad suficiente para evaluación práctica y teórica, orientada a la resolución de problemas reales en la infraestructura informática de EMTUSA. Se aplican las normativas vigentes: RGPD y ENS.
