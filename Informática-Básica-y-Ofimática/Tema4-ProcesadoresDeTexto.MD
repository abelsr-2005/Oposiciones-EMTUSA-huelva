# 📝 TEMA 4: Procesadores de texto – Word 365

## 📘 Introducción

Microsoft **Word 365** es uno de los procesadores de texto más utilizados dentro del paquete **Microsoft 365**. Permite crear, editar, dar formato e imprimir documentos de texto de forma profesional.

Entre sus funciones principales se encuentran la redacción de documentos, la inserción de imágenes y tablas, la aplicación de estilos de formato, la revisión ortográfica y la preparación de documentos para su impresión o distribución digital.

Dominar Word es fundamental para la creación y organización de documentos en entornos académicos y profesionales.

---

# ⚙️ 4.1 Principales funciones y utilidades

Word 365 utiliza una interfaz basada en la **Cinta de opciones (Ribbon)**, que organiza las herramientas en pestañas para facilitar el acceso a las distintas funciones.

Entre las pestañas más importantes se encuentran:

* **Inicio** → contiene herramientas básicas de edición y formato de texto.
* **Insertar** → permite añadir elementos como imágenes, tablas, gráficos o encabezados.
* **Disposición** → controla márgenes, orientación de página y tamaño del documento.
* **Referencias** → incluye herramientas para crear índices, citas o tablas de contenido.

Esta organización permite acceder de forma rápida a las funciones más utilizadas.

---

## 📄 Formatos de archivo de Word

Word utiliza distintos formatos de archivo para guardar documentos.

Los más importantes son:

**.docx**
Es el formato estándar actual de Word. Está basado en tecnología **XML**, lo que permite archivos más ligeros y seguros.

**.docm**
Documento de Word que contiene **macros**. Las macros son pequeños programas que automatizan tareas dentro del documento.

**.dotx**
Archivo de **plantilla de Word** que permite crear documentos nuevos con un diseño predefinido.

**.doc**
Formato antiguo utilizado en versiones de Word anteriores a 2007.

---

# 📑 4.2 Creación y estructuración del documento

Para crear documentos bien organizados es importante utilizar las herramientas de estructuración que ofrece Word.

---

## 🎨 Estilos

Los **estilos** permiten aplicar formatos predefinidos al texto, como títulos, subtítulos o texto normal.

Ejemplos de estilos:

* Título 1
* Título 2
* Título 3
* Normal

El uso de estilos ofrece varias ventajas:

* mantiene una apariencia uniforme en todo el documento
* permite modificar el formato global fácilmente
* facilita la creación automática de **tablas de contenido**

---

## ⌨️ Saltos de página

Un **salto de página** fuerza que el contenido siguiente comience en una nueva página.

Atajo de teclado:

**Ctrl + Enter**

Esto permite organizar correctamente capítulos o apartados dentro del documento.

---

## ⚠️ Diferencia entre salto de página y salto de sección

Es una diferencia muy frecuente en preguntas de examen.

**Salto de página**
Simplemente hace que el texto continúe en la página siguiente.

**Salto de sección**
Divide el documento en secciones independientes.

Esto permite que cada sección tenga configuraciones diferentes, por ejemplo:

* orientación vertical u horizontal
* encabezados distintos
* numeración de páginas diferente

---

# 💾 4.3 Gestión, grabación, recuperación e impresión

Word permite guardar, recuperar e imprimir documentos mediante distintas herramientas.

---

## 💾 Guardar documentos

Existen dos formas principales de guardar un documento.

**Guardar (Ctrl + G)**
Actualiza el documento actual sobrescribiendo los cambios realizados.

**Guardar como (F12)**
Permite crear una copia del documento o cambiar:

* el nombre del archivo
* la ubicación de almacenamiento
* el formato del archivo

Por ejemplo, un documento puede exportarse como **PDF**.

---

## 🔄 Recuperación de documentos

Word incluye sistemas automáticos para evitar la pérdida de información.

**Autorrecuperación**
Guarda automáticamente copias temporales del documento cada cierto tiempo.

Si el programa se cierra inesperadamente, Word puede recuperar la última versión guardada.

**Autoguardado**
Función disponible cuando el documento está almacenado en la nube mediante **OneDrive o SharePoint**.

En este caso los cambios se guardan automáticamente en tiempo real.

---

## 🖨️ Impresión

Para imprimir un documento se utiliza el comando:

**Ctrl + P**

Desde el menú de impresión se pueden configurar diferentes opciones:

* número de copias
* impresora
* orientación de la página
* márgenes
* páginas a imprimir

---

## ⚠️ Intercalar copias

Cuando se imprimen varias copias de un documento de varias páginas existen dos opciones.

**Intercalar**
Imprime los documentos completos en orden.

Ejemplo (3 páginas, 3 copias):

1-2-3 / 1-2-3 / 1-2-3

**Sin intercalar**
Imprime todas las copias de cada página antes de pasar a la siguiente.

Ejemplo:

1-1-1 / 2-2-2 / 3-3-3

# 🎛️ 4.4 Personalización del entorno de trabajo

Word permite adaptar su interfaz para facilitar el trabajo del usuario.

---

## ⭐ Barra de herramientas de acceso rápido

La **barra de herramientas de acceso rápido** se sitúa normalmente en la parte superior izquierda de la ventana.

Permite añadir comandos utilizados con frecuencia, como:

* Guardar
* Deshacer
* Rehacer

Esto facilita acceder rápidamente a funciones importantes.

---

# 👁️ Vistas del documento

Word ofrece diferentes formas de visualizar el documento según la tarea que se esté realizando.

---

## 🖨️ Diseño de impresión

Es la vista predeterminada.

Muestra el documento exactamente como aparecerá al imprimirse, incluyendo márgenes, encabezados y saltos de página.

---

## 🧭 Vista Esquema

Organiza el documento como una **estructura jerárquica basada en los estilos de título**.

Esta vista permite:

* reorganizar capítulos
* mover secciones completas
* visualizar la estructura del documento

---

## ✏️ Vista Borrador

Oculta elementos de diseño como:

* encabezados
* pies de página
* márgenes

Esto permite trabajar más rápido cuando se editan documentos muy largos.

---

Ctrl + G → guardar documento.

F12 → guardar como.

Ctrl + P → imprimir documento.
