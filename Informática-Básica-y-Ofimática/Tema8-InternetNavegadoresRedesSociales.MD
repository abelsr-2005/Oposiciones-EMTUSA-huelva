# 🌐 TEMA 8: Internet, Navegadores y Redes Sociales

## 📘 Introducción

Internet es una red global que conecta millones de dispositivos en todo el mundo y permite el intercambio de información y la comunicación entre usuarios. Gracias a esta red es posible acceder a páginas web, enviar correos electrónicos, utilizar servicios en línea y participar en redes sociales.

Para acceder a los contenidos de Internet se utilizan **navegadores web**, que permiten visualizar páginas y utilizar servicios en línea. Además, el uso de Internet requiere comprender aspectos básicos de **seguridad digital**, especialmente al interactuar con servicios web y redes sociales.

---

# 🌍 8.1 Conceptos fundamentales de Internet

Internet es una **red mundial de redes interconectadas** que utilizan el conjunto de protocolos **TCP/IP** para comunicarse entre sí.

Gracias a esta infraestructura es posible transmitir datos entre ordenadores situados en cualquier parte del mundo.

Internet ofrece múltiples servicios, entre los que destaca la **World Wide Web (WWW)**, que permite acceder a páginas web mediante navegadores.

---

## 🔗 Protocolos y servicios de Internet

### HTTP y HTTPS

El **HTTP (HyperText Transfer Protocol)** es el protocolo utilizado para transferir páginas web entre el servidor y el navegador del usuario.

El **HTTPS** es la versión segura de este protocolo. Utiliza cifrado mediante **TLS/SSL** para proteger la comunicación entre el usuario y el servidor.

Este cifrado evita que terceros puedan interceptar la información transmitida.

---

### DNS

El **DNS (Domain Name System)** es el sistema encargado de traducir los nombres de dominio en direcciones IP.

Por ejemplo:

[www.google.com](http://www.google.com) → 142.250.xxx.xxx

Gracias al DNS los usuarios pueden acceder a páginas web mediante nombres fáciles de recordar en lugar de direcciones numéricas.

---

### URL

Una **URL (Uniform Resource Locator)** es la dirección que identifica un recurso en Internet.

Su estructura básica es:

protocolo://dominio:puerto/ruta?parámetros

Ejemplo:

[https://www.ejemplo.com/index.html](https://www.ejemplo.com/index.html)

---

### World Wide Web (WWW)

La **World Wide Web** es un servicio de Internet que permite acceder a documentos de hipertexto enlazados entre sí.

Es importante recordar que:

* **Internet** es la infraestructura de red.
* **WWW** es uno de los servicios que funciona sobre esa red.

---

# 🌐 8.2 Navegadores web

Los **navegadores web** son programas que permiten acceder y visualizar páginas de Internet.

Entre los navegadores más utilizados se encuentran:

* Microsoft Edge
* Google Chrome

Ambos navegadores están basados actualmente en el proyecto **Chromium**, lo que hace que utilicen tecnologías similares para interpretar páginas web.

---

## ⚙️ Gestión de datos locales en el navegador

Cuando se navega por Internet, el navegador almacena cierta información en el equipo del usuario.

### Caché

La **caché** guarda elementos de las páginas web, como imágenes o scripts, para que se carguen más rápido en futuras visitas.

Esto reduce el consumo de ancho de banda y mejora la velocidad de navegación.

---

### Cookies

Las **cookies** son pequeños archivos de datos que los sitios web almacenan en el navegador del usuario.

Se utilizan para:

* mantener sesiones abiertas
* recordar preferencias
* personalizar contenidos
* realizar seguimiento de navegación

---

### Historial

El **historial de navegación** es un registro de las páginas web visitadas.

Permite volver fácilmente a sitios visitados anteriormente.

---

## 🕶️ Modo incógnito o InPrivate

Los navegadores incluyen un modo de navegación privada.

En **Chrome** se denomina **modo incógnito** y en **Edge** se llama **InPrivate**.

Este modo evita que se guarden en el equipo:

* historial de navegación
* cookies
* datos de formularios

Sin embargo, no garantiza el anonimato total, ya que:

* el proveedor de Internet (ISP)
* los administradores de red

pueden seguir viendo la actividad de navegación.

---

# 🔐 8.3 Seguridad y certificados digitales

La seguridad en Internet es fundamental para proteger la información transmitida entre usuarios y servidores.

---

## 📜 Certificados digitales SSL/TLS

Un **certificado SSL/TLS** es un documento digital que verifica la identidad de un servidor web y permite establecer una conexión cifrada.

Gracias a estos certificados es posible proteger la información intercambiada entre el navegador y el servidor.

Los sitios que utilizan este sistema muestran el prefijo **https** y un icono de candado en el navegador.

---

## 🏛️ Entidades de certificación

Las **Autoridades de Certificación (CA)** son organizaciones de confianza encargadas de emitir certificados digitales.

Un ejemplo en España es la **FNMT (Fábrica Nacional de Moneda y Timbre)**.

Estas entidades garantizan que el certificado pertenece realmente a la organización que lo utiliza.

---

## ✍️ Firma electrónica

La **firma electrónica** es un mecanismo criptográfico que permite verificar la identidad del emisor de un documento digital.

Garantiza tres aspectos fundamentales:

* **autenticidad** → confirma quién envía el documento
* **integridad** → asegura que el contenido no ha sido modificado
* **no repudio** → el emisor no puede negar haber enviado el documento

---

# 📱 8.4 Redes sociales y seguridad

Las **redes sociales** son plataformas digitales que permiten la comunicación, la difusión de información y la interacción entre usuarios.

En entornos profesionales también se utilizan como herramientas de comunicación institucional.

---

## 💼 Redes sociales profesionales

Algunas de las plataformas más utilizadas son:

**LinkedIn**
Red social orientada al ámbito profesional y al networking laboral.

**X (Twitter)**
Plataforma utilizada para comunicación rápida y difusión de información institucional.

**Instagram y Facebook**
Redes sociales centradas en contenido visual y comunicación con comunidades de usuarios.

---

## ⚠️ Riesgos de seguridad en redes sociales

El uso de redes sociales también puede implicar riesgos de seguridad.

### Phishing

El **phishing** es una técnica de suplantación de identidad utilizada para robar credenciales o información personal.

Los atacantes suelen enviar enlaces falsos que imitan páginas legítimas.

---

### Ingeniería social

La **ingeniería social** consiste en manipular a las personas para que revelen información confidencial.

No se basa en fallos técnicos, sino en la confianza del usuario.

---

### Malware

El **malware** es software malicioso diseñado para dañar sistemas o robar información.

Puede distribuirse mediante:

* enlaces maliciosos
* archivos adjuntos
* perfiles falsos en redes sociales

---

## ⚠️ Punto importante para examen

En el **modo incógnito**, el navegador no guarda el historial ni las cookies.

Sin embargo:

* **los archivos descargados permanecen en el equipo**
* **los marcadores guardados siguen existiendo**

---

# 🧠 Resumen de conceptos clave

Internet → red global basada en protocolos TCP/IP.

HTTP / HTTPS → protocolos de transferencia de páginas web.

DNS → traduce nombres de dominio en direcciones IP.

URL → dirección de un recurso en Internet.

Navegador → programa para acceder a páginas web.

Caché → almacena elementos para acelerar la carga.

Cookies → archivos que guardan información de navegación.

Certificados SSL/TLS → permiten comunicaciones cifradas.

Firma electrónica → garantiza autenticidad e integridad.

Phishing → suplantación de identidad para robar datos.

Malware → software malicioso distribuido en la red.
