# TEMA 2: Introducción al sistema operativo: el entorno Windows

> **Objetivo del examen:** Dominar los fundamentos del entorno gráfico de Windows, la gestión de ventanas, iconos, menús contextuales y cuadros de diálogo. Conocer en profundidad el escritorio, sus elementos, el menú de inicio y las funcionalidades de Cortana.

## 2.1. Fundamentos y Entorno Gráfico de Windows
* Windows utiliza una **Interfaz Gráfica de Usuario (GUI)**, que permite al usuario interactuar con el sistema operativo mediante imágenes y objetos gráficos en lugar de comandos de texto.

## 2.2. Trabajo en el entorno gráfico: Ventanas, Iconos y Cuadros de Diálogo
* **Ventanas:** Áreas rectangulares de la pantalla donde se muestran las aplicaciones o carpetas. Poseen controles estándar en la barra de título: Minimizar, Maximizar/Restaurar y Cerrar.
* **Iconos:** Representaciones gráficas de programas, archivos, carpetas o accesos directos.
* **Cuadros de Diálogo:** Son ventanas secundarias que el sistema o una aplicación abre para solicitar información al usuario o pedirle que confirme una acción (ej. "Guardar como" o "Imprimir").

### ⚠️ Alerta de Trampa (Ventanas vs Cuadros de Diálogo)
En los test, la principal diferencia técnica que buscan es que un cuadro de diálogo **suele carecer de botones para maximizar o minimizar**, y requiere obligatoriamente una interacción del usuario (aceptar, cancelar, introducir datos) para poder continuar trabajando en la aplicación principal.

## 2.3. Menús Contextuales
* Son menús emergentes que muestran una lista de comandos directamente relacionados con el objeto seleccionado o el área donde se hace clic.
* Se activan haciendo **clic con el botón derecho del ratón** sobre un elemento o utilizando el atajo de teclado `Shift + F10` (o la tecla dedicada en el teclado físico).

## 2.4. El Escritorio y sus elementos
* **El Escritorio:** Es el área principal de trabajo en pantalla que aparece después de iniciar sesión en Windows.
* **Elementos principales:**
    1. **Fondo de escritorio** (Wallpaper).
    2. **Iconos y Accesos directos** (los accesos directos se distinguen visualmente por una pequeña flecha en la esquina inferior izquierda del icono). Si eliminas el acceso directo, el archivo original no se ve afectado.
    3. **Barra de Tareas:** Generalmente ubicada en la parte inferior. Contiene el botón de Inicio, iconos de aplicaciones ancladas, aplicaciones en ejecución y el Área de Notificación (System Tray) a la derecha, donde se aloja el reloj y los iconos de estado (red, volumen, etc.).

## 2.5. El menú inicio
* Es la puerta de entrada principal a los programas, carpetas y configuración del equipo.
* Se abre haciendo clic en el icono de Windows en la barra de tareas o pulsando la tecla `Windows` (o `Ctrl + Esc`).

### ⚠️ Alerta de Trampa (El Menú de Usuario Avanzado)
Pregunta clásica para administradores de sistemas: Si haces **clic derecho** sobre el botón de Inicio (o pulsas el atajo `Windows + X`), no se abre el menú inicio estándar, sino un menú contextual avanzado con accesos directos a herramientas de administración críticas (PowerShell/CMD como Administrador, Administración de discos, Visor de eventos, etc.).

## 2.6. Cortana
* Es el **asistente de productividad personal y virtual** de Microsoft.
* Permite la interacción mediante **comandos de voz** o texto en lenguaje natural.
* Funciones clave: Realizar búsquedas locales (archivos) y en internet, configurar recordatorios, gestionar el calendario y abrir aplicaciones.

### ⚠️ Alerta de Trampa (Cortana)
Aunque Microsoft ha ido retirando Cortana en favor de Copilot en las versiones más recientes de Windows 11, el temario la incluye explícitamente. A nivel de test, debes identificarla siempre como el "asistente virtual" o "asistente de voz" originario del ecosistema Windows 10.
