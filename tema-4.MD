# TEMA 4: Procesadores de texto: Word 365

> **Objetivo del examen:** Dominar las principales funciones y utilidades de Word 365, la creación y estructuración del documento, la gestión, grabación, recuperación e impresión de ficheros y la personalización del entorno de trabajo.

## 4.1. Principales funciones y utilidades
* **Interfaz principal:** Word 365 utiliza la Cinta de opciones, que organiza las herramientas en pestañas lógicas (Inicio, Insertar, Disposición, Referencias).
* **Extensiones de archivo (Visión IT):** El tribunal exigirá conocer los formatos nativos.
    * .docx: Formato estándar predeterminado basado en XML.
    * .docm: Documento de Word habilitado para macros (contiene código ejecutable, crucial a nivel de seguridad).
    * .dotx: Plantilla de Word.
    * .doc: Formato heredado (versiones Word 97-2003).

## 4.2. Creación y estructuración del documento
* **Estilos:** Es la herramienta más importante para la estructuración técnica del documento. Aplicar estilos (Título 1, Título 2, Normal) permite mantener la coherencia visual, cambiar el diseño global de forma centralizada y generar Tablas de Contenido automáticas.
* **Saltos de página (Ctrl + Enter):** Fuerza el inicio del texto en la página siguiente, manteniendo el formato general intacto.

### 🚸 Alerta de Trampa (Estructuración: Página vs Sección)
Pregunta de examen garantizada: Un Salto de Página se limita a pasar el texto a la hoja siguiente. Sin embargo, un **Salto de Sección** aísla el documento en partes independientes, permitiendo que cada sección tenga una configuración de página totalmente distinta (ej. una página en orientación vertical y la siguiente en horizontal, o encabezados diferentes).

## 4.3. Gestión, grabación, recuperación e impresión
* **Grabación (Guardado):**
    * Guardar (Ctrl + G): Sobrescribe el archivo actual.
    * Guardar como (F12): Permite crear una copia, cambiar el nombre, modificar la ubicación o alterar el formato de salida (ej. exportar a PDF).
* **Recuperación de ficheros:**
    * Autorrecuperación: Guarda copias de seguridad temporales en el almacenamiento local cada X minutos para prevenir pérdida de datos.
    * Autoguardado: Función exclusiva del ecosistema Microsoft 365 que guarda los cambios en tiempo real, pero **solo opera si el archivo está alojado en la nube** (OneDrive o SharePoint).
* **Impresión (Ctrl + P):** Permite configurar copias, orientación, márgenes y rangos de páginas.
    * Trampa habitual (Intercalar): Si imprimes 3 copias de un documento de 3 páginas, la opción "Intercalar" imprime bloques ordenados (1-2-3, 1-2-3, 1-2-3). La opción "Sin intercalar" imprime por lotes de página (1-1-1, 2-2-2, 3-3-3).

## 4.4. Personalización del entorno de trabajo
* **Barra de herramientas de acceso rápido:** Situada tradicionalmente en la esquina superior izquierda, permite al usuario anclar comandos frecuentes (Guardar, Deshacer) para que estén siempre accesibles.
* **Vistas del documento:** * Diseño de impresión: Vista predeterminada; renderiza el documento tal y como se imprimirá físicamente.
    * Esquema: Muestra el documento como un árbol jerárquico basado en los Estilos de Título. Herramienta definitiva para mover capítulos enteros.
    * Borrador: Oculta encabezados, pies de página y márgenes para maximizar el rendimiento al editar grandes volúmenes de texto plano.
