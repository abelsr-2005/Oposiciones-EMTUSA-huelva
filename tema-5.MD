# TEMA 5: Hojas de cálculo: Excel 365

> **Objetivo del examen:** Dominar las principales funciones y utilidades de Excel 365, la estructura de libros y hojas, la edición de datos, el uso de fórmulas y funciones, y la gestión de gráficos.

## 5.1. Principales funciones y utilidades
* Aplicación de hojas de cálculo orientada al procesamiento de datos numéricos y alfanuméricos organizados en tablas.
* **Extensiones de archivo (Perspectiva IT):**
    * .xlsx: Formato predeterminado basado en el estándar XML.
    * .xlsm: Libro habilitado para macros (contiene código VBA), crítico para la seguridad del endpoint.
    * .csv: Formato de texto plano con valores separados por comas, esencial para la interoperabilidad de datos.

## 5.2. Libros, hojas y celdas
* **Libro:** El contenedor o archivo principal (.xlsx) que puede albergar múltiples hojas de cálculo.
* **Hoja de cálculo:** Matriz bidimensional donde las columnas se identifican por letras y las filas por números.
* **Celda:** La unidad mínima de trabajo, definida por la intersección de una fila y una columna.
* **Rango:** Selección de dos o más celdas (ejemplo A1:C5).

### ⚠️ Alerta de Trampa (Comandos de edición)
* **F2:** Activa el modo de edición de la celda seleccionada sin borrar el contenido previo.
* **Ctrl + Inicio:** Desplaza el foco del cursor inmediatamente a la celda A1.
* **F4:** Durante la edición de una fórmula, permite alternar cíclicamente entre los tipos de referencias.

## 5.3. Introducción y edición de datos
* Excel interpreta el contenido según su naturaleza: texto, valores numéricos, fechas o funciones.
* **Formato de celdas:** Permite definir la representación visual de los datos (moneda, porcentaje, fecha, contabilidad) sin alterar el valor real almacenado.

## 5.4. Fórmulas y funciones
* **Fórmulas:** Expresiones de cálculo creadas por el usuario. Deben comenzar siempre con el signo `=`.
* **Funciones:** Estructuras de cálculo predefinidas por el software para operaciones complejas (ej. BUSCARV, SI, SUMA).

### ⚠️ Alerta de Trampa (Tipos de Referencias)
Este es el concepto más evaluado en los exámenes de informática de oposiciones:
1. **Referencias Relativas (A1):** Se ajustan automáticamente al copiar o arrastrar la celda a una nueva posición.
2. **Referencias Absolutas ($A$1):** El símbolo `$` bloquea la fila y la columna, impidiendo que la referencia cambie al desplazar la fórmula.
3. **Referencias Mixtas ($A1 o A$1):** Solo una de las coordenadas (fila o columna) permanece fija.

## 5.5. Gráficos y Gestión de datos
* **Gráficos:** Herramientas de representación visual para el análisis de tendencias (barras, líneas, circulares, dispersión).
* **Herramientas de tabla:**
    * **Ordenar:** Reorganización de registros bajo criterios alfabéticos o numéricos.
    * **Filtrar:** Segmentación lógica de los datos para mostrar solo aquellos que cumplen criterios específicos.

## 5.6. Personalización del entorno de trabajo
* **Inmovilizar paneles:** Opción de la pestaña 'Vista' que permite fijar filas o columnas (típicamente cabeceras) para que permanezcan visibles durante el desplazamiento por grandes conjuntos de datos.
* **Modos de Visualización:** Normal, Diseño de página y Vista previa de salto de página.
